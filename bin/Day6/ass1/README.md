# Flutter Food Delivery App - Learning Project

This repository contains a simple **Flutter** app for food delivery, developed as a learning project. The code demonstrates fundamental Flutter concepts such as routing, navigation, and UI widgets.

## Features

- Splash screen on startup
- Home screen showcasing categories
- Categories, Cart, Delivery, and Profile screens
- Simple routing between different screens/pages

## Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- Dart
- An editor like VS Code or Android Studio

### Running the App

1. Clone the repository:
   ```
   git clone <REPOSITORY_URL>
   ```
2. Change to the project directory:
   ```
   cd <project_folder>
   ```
3. Get the dependencies:
   ```
   flutter pub get
   ```
4. Run the app:
   ```
   flutter run
   ```

## Project Structure

- `lib/main.dart` - Entry point and main routing
- `lib/home_screen.dart` - Home screen
- `lib/splash_screen.dart` - Splash screen
- Other screens (Categories, Cart, Deliver, Profile)
