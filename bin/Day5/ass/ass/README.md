# Flutter Movie App

This project is a simple Flutter application created for learning purposes. It demonstrates the basic setup of a Flutter app with a dark theme and navigation structure. The main screen of the application is a movie screen, which you can further expand to display lists or details about movies.

## Features

- **Dark Mode**: The app uses a dark-themed design for a modern look.
- **Bottom Navigation Bar**: Includes a persistent bottom navigation bar with customization for selected/unselected items.
- **Scaffold Background**: Custom background colors for a polished appearance.
- **Easy Customization**: The code is structured for easy learning and editing.

## Getting Started

To run this project on your local machine:

1. **Clone the repository**

   ```
   git clone <repository-url>
   ```

2. **Install dependencies**

   ```
   flutter pub get
   ```

3. **Run the app**

   ```
   flutter run
   ```

## Folder Structure

- `lib/main.dart`: Main entry point of the application.
- `lib/movie_screen.dart`: The main screen (placeholder to be implemented as you learn).
- `assets/`: (Optionally) Place images or resource files here, as referenced in `pubspec.yaml`.

## Requirements

- Flutter SDK 3.7.2 or higher (see [`pubspec.yaml`](pubspec.yaml) for details)

## Notes

- This project is intended for educational purposes and does not contain complex features.
- You are encouraged to expand and customize the app as you learn more about Flutter.
